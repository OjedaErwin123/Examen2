# Calculadora de MRU y MRUA

Esta es mi calculadora para MRU y MRUA

La cual te permite calcular diferentes aspectos del Movimiento Rectilíneo Uniforme (MRU) y el Movimiento Rectilíneo Uniformemente Acelerado (MRUA) de manera rápida y sencilla.

## Cómo Probar

1. **Ingresa al repositorio:** Ingresa a la carpeta codigo.

2. **Ejecuta el Program.cs:** Dentro de la carpeta estara un ejecutable, copia el codigo y pruebalo! para empezar a usar la calculadora

3. **Usar la Calculadora:**
   - Ingresa los valores de velocidad inicial, velocidad final, tiempo y aceleración cuando se te solicite.
   - Selecciona qué componente del movimiento deseas calcular.
   - ¡Obtén tu resultado!
